05 — UNIFIED CLIENT EXPERIENCE

One Library, Any Model, Either Protocol

from triton_client import TritonGrpcClient

client = TritonGrpcClient(
    url="localhost:50051",
    api_key=os.environ["DOMINO_USER_API_KEY"]
)

# YOLOv8n — Video
result = client.infer(
    model="yolov8n",
    inputs={"images": frame_tensor},
    outputs=["output0"]
)

# BERT — Text
result = client.infer(
    model="bert-base-uncased",
    inputs={
        "input_ids": ids,
        "attention_mask": mask
    },
    outputs=["logits"]
)

# Same pattern for any model!
Consistent Interface
Same client.infer() call regardless of model type. Pass numpy arrays in, get numpy arrays out.
Protocol Transparent
Swap TritonGrpcClient for TritonRestClient — zero code changes.
Auto-Handled Complexity
Dtype mapping, proto construction, response parsing — all automatic. 80% code reduction.
Built-in Auth
Domino API key or JWT at construction. Every request authenticated automatically.
